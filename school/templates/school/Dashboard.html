I- Les importations

Le menu des importations vous permet de renseigner votre base de données à travers des fichiers Excel aux formats précis. Si vous avez des données existantes. Il est important de respecter l'ordre des importations à savoir:
* Importer les niveaux et les degrés d'enseignement
* Importer les salles de classes
* Importer les apprenants

Pour éffectuer une importation, accedez au sous-menu de l'importation que vous souhaitez réaliser et suivez les étapes comme suit:
1- Téléchargez le fichier témoin à remplir
2- Cliquez sur le boutton ajouter pour éffectuer une nouvelle importation
3- Sélectionnez le fichier rempli à importer
4- Cliquez sur le boutton enregistrer pour éffectuer l'importation

II- Les salles de classes

Le sous-menu des salles de classes vous permet d'enregistrer et de consulter les différents niveaux et degrés d'enseignement disponibles dans votre établissement, et ainsi que les différentes salles de classes dont vous disposez.

II-1- Les niveaux d'enseignement

Pour éffectuer un enregistrement d'un niveaux d'enseignement, accedez au sous-menu Niveaux et degrés / Niveaux et suivez les étapes comme suit:
1- Cliquer sur le boutton ajouter pour éffectuer un nouvel enregistrement
2- Remplissez les informations obligatoires
3- Cliquer sur le boutton enregistrer pour éffectuer l'enregistrement

II-2- Les degrés d'enseignement

Pour éffectuer un enregistrement d'une classe, accedez au sous-menu Niveaux et degrés / Degrés et suivez les étapes comme suit:
1- Cliquer sur le boutton ajouter pour éffectuer un nouvelle enregistrement
2- Remplissez les informations obligatoires
3- Cliquer sur le boutton enregistrer pour éffectuer l'enregistrement

II-3- Les salles de classes

Pour éffectuer un enregistrement d'une classe, accedez au sous-menu Consulter / Créer et suivez les étapes comme suit:
1- Cliquer sur le boutton ajouter pour éffectuer un nouvel enregistrement
2- Remplissez les informations obligatoires
3- Cliquer sur le boutton enregistrer pour éffectuer l'enregistrement

III- Les apprenants

Le menu des apprenants vous permet de consulter et d'enregistrer les informations des différents apprennants de votre etablissement.

Pour éffectuer un enregistrement d'un apprenant, accedez au menu Consulter / Créer et suivez les étapes comme suit:
1- Cliquer sur le boutton ajouter pour éffectuer un nouvel enregistrement
2- Remplissez les informations obligatoires
3- Cliquer sur le boutton enregistrer pour éffectuer l'enregistrement

IV- Les tuteurs

Le menu des tuteurs vous permet de consulter et d'enregistrer les informations des différents tuteurs de votre etablissement et aussi de leur envoyer des messages.

IV-1- Les Tuteurs

Pour éffectuer un enregistrement d'un tuteur, accedez au menu Consulter / Créer et suivez les étapes comme suit:
1- Cliquer sur le boutton ajouter pour éffectuer un nouvel enregistrement
2- Remplissez les informations obligatoires
3- Cliquer sur le boutton enregistrer pour éffectuer l'enregistrement

IV-2- Les messages aux tuteurs

Avant de pouvoir envoyer des messages aux tuteurs, vous devez tout d'abord configurer les paramètres des comptes Email et Sms par lesquels vous passerez. Accedez au menu EMAIL / SMS du module Administration->Etablissement

IV-2-A- Email


Pour éffectuer l'enregistrement de la configuration Email , accedez au menu EMAIL / SMS et suivez les étapes comme suit:
1- Cliquer sur le boutton modifier pour éffectuer l'enregistrement
2- Remplissez les informations obligatoires
3- Cliquer sur le boutton enregistrer pour éffectuer l'enregistrement

IV-2-B- SMS


Pour éffectuer l'enregistrement de la configuration SMS , accedez au menu EMAIL / SMS et suivez les étapes comme suit:
1- Cliquer sur le boutton modifier pour éffectuer l'enregistrement
2- Remplissez les informations obligatoires
3- Cliquer sur le boutton enregistrer pour éffectuer l'enregistrement

IV-2-C- Envoyer un message


Pour envoyer un message aux tuteurs, accedez au menu Les messages et suivez les étapes comme suit:
1- Cliquer sur le boutton ajouter pour éffectuer l'envoi
3- Remplissez les informations obligatoires
4- Cliquer sur le boutton enregistrer pour éffectuer l'envoi

V- Les inscriptions

Le sous-menu des inscriptions vous permet d'enregistrer et de consulter les insriptions des apprenants dans votre établissement.

Pour éffectuer un enregistrement d'une inscription, accedez au sous-menu Consulter / Créer et suivez les étapes comme suit:
